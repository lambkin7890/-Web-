运行环境：
	node+npm
	安装方式：https://www.liaoxuefeng.com/wiki/1022910821149312/1023025597810528
	淘宝镜像安装命令npm install -g cnpm --registry=https://registry.npm.taobao.org

运行方式：
	cmd进入项目目录下
	1.cnpm install
	2.cnpm run serve
卸载node_modules:
	安装rimraf:  cnpm install rimraf -g

  	删除node_modules:  rimraf node_modules
